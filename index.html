Browser Info

This browser uses an iframe to load websites. Many modern websites block iframe embedding due to security headers.

Only websites that do not restrict iframe usage (usually non-profit or developer-friendly sites) will display properly.

Blocked sites use HTTP headers like:

  • X-Frame-Options: DENY
  • Content-Security-Policy: frame-ancestors 'none';
